# Expense Tracker 💰

A simple Expense Tracker web application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add income and expense transactions
- Automatically calculates balance
- Displays income and expense summary
- Simple and clean UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📌 How to Use
1. Enter transaction description
2. Enter amount (positive for income, negative for expense)
3. Click "Add Transaction"
4. View updated balance instantly
